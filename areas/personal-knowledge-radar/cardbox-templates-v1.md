# 个人知识卡片盒模板 v1

状态：working
更新时间：2026-05-13
关联：[[personal-knowledge-cardbox-v1]]

## 使用原则

所有外部材料先进入 inbox，不直接变成长文总结。

每条材料只做最小接入：
- 固定来源
- 选一个主领域
- 标一个状态
- 至少连到 1 个已有卡片或项目
- 明确下一步动作

主领域只允许五选一：
- 投资理财
- AI 效率
- 自媒体运营
- 关系心理
- 生活视野

生活视野是探索池、杂项池、新领域孵化池；如果某类内容连续出现高价值，再从生活视野里孵化成新领域。

## 状态字段 PARA 化

- inbox：刚收进来，未判断
- working：正在加工或本周可能使用
- tool：已经转成可复用工具、模板、Skill、方法卡
- archive：归档、低频、已完成、过时或仅保留证据

## 卡片类型

### 1. 来源卡模板

用途：管理公众号、网站、作者、数据源，不做文章总结。

标题：来源卡 - 名称

字段：
- source_name：
- source_type：公众号 / 网站 / 作者 / 数据源 / 工具 / 其他
- exact_id：
- verified：yes / no / pending
- source_trust：high / medium / low
- whitelist_status：active_full_corpus / active_tracking / candidate_watch / borrow_study / archive
- 主领域：投资理财 / AI 效率 / 自媒体运营 / 关系心理 / 生活视野
- 标签：
- 适合学习：
- 不适合学习：
- 代表材料：
- 下一步：继续追踪 / 补样本 / 降级观察 / 归档
- 链接到：

最小判断：这个来源长期值得占用注意力吗？

- 偏好度：1-5
- 当前状态：inbox / working / tool / archive

## 每日行动卡选择规则

每天最多推 1-3 张行动卡。

优先级计算不复杂化，按四个维度人工判断：
- 重要度：长期是否有用
- 偏好度：foras 现在是否愿意做
- 可转化度：能不能变成内容、工具、Skill、投研判断或关系判断
- 精力匹配：今天状态能不能完成

默认排序：
高重要 + 高偏好 + 低摩擦 + 有明确产出。

## 每周复盘规则

每周只问五个问题：
- 投资理财有没有形成一个更好的判断？
- AI 效率有没有让 Hermes 或工具链更强？
- 自媒体运营有没有新增选题、表达或产品资产？
- 关系心理有没有沉淀一个机制、边界或案例判断？
- 生活视野有没有出现值得升级的新方向？

## 不做什么

- 不把每篇文章都总结成长文。
- 不追求完美目录。
- 不让生活视野污染主线。
- 不把工具箱越做越大而没有行动。
- 不把每日推送做成信息流。
