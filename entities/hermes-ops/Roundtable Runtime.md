# Roundtable Runtime

## 是什么
Roundtable Runtime 是组织人物发言、证据装载、独立首轮、交叉质疑、结果停点与会议记录输出的运行时媒介层。

## 在本系统中的角色
- 它不是人物主体，而是人物工作的会议媒介
- 它不拥有判断权，只负责组织、装载、记录、停点
- 它与股票研究证据页分离，归属于 Hermes / Agent Ops Wiki
